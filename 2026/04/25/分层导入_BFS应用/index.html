еҲҶеұӮеҜје…ҘпјҡBFS еңЁеҲҶеұӮеҜје…Ҙдёӯзҡ„е®һи·ө

Joie2026-04-252026-04-25

иғҢжҷҜдёҺзӣ®ж Ү

йңҖиҰҒе®һзҺ°дёҖдёӘвҖңеҲҶеұӮеҜје…ҘвҖқеҠҹиғҪпјҡд»Һз¬¬дёүж–№е№іеҸ°пјҲзҹҘиҜҶеә“/ж–Үд»¶еӨ№/ж–Үд»¶пјүеөҢеҘ—еҜје…ҘеҲ°жң¬ең°е№іеҸ°пјҢе№¶е®Ңж•ҙдҝқз•ҷз¬¬дёүж–№зҡ„еұӮж¬Ўз»“жһ„дёҺе…ғж•°жҚ®гҖӮзӣ®ж ҮжҳҜпјҡ

  • дҝқжҢҒзӣ®еҪ•еұӮзә§дёҖиҮҙ
  • жү№йҮҸй«ҳж•Ҳе…Ҙеә“пјҲеҮҸе°‘еӨҡж¬ЎеҫҖиҝ”дёҺ ID з«һдәүпјү
  • дёҺеӨ§ж–Үд»¶дёӢиҪҪ/дёҠдј и§ЈиҖҰпјҢејӮжӯҘжҺЁиҝӣе№¶еҸҜи§ӮжөӢиҝӣеәҰ

жҖ»дҪ“жөҒзЁӢпјҲBFSпјү

ж•ҙдҪ“жөҒзЁӢеҸҜжҰӮжӢ¬дёәпјҡ

  • з”ЁжҲ·йҖүжӢ©ж №иҠӮзӮ№пјҲзҹҘиҜҶеә“/ж–Үд»¶еӨ№/ж–Үд»¶пјү
  • дҪҝз”ЁйҳҹеҲ—жҢүеұӮпјҲBFSпјүйҒҚеҺҶз¬¬дёүж–№з»“жһ„пјҢжһ„е»әжң¬ең° Folder/Dataset е®һдҪ“
  • жү№йҮҸйў„еҲҶй…Қ IDпјҢдҝ®жӯЈзҲ¶еӯҗеј•з”Ё
  • жҢүеұӮжү№йҮҸе…Ҙеә“пјҢжіЁеҶҢејӮжӯҘд»»еҠЎ
  • йҖҡиҝҮж¶ҲжҒҜйҳҹеҲ—и§ҰеҸ‘дёӢиҪҪ+дёҠдј пјҢRedis е®һж—¶и®°еҪ•иҝӣеәҰ

жөҒзЁӢеӣҫ

д»Јз Ғе®һзҺ°

жң¬иҠӮеҢ…еҗ«пјҡ

  • е…ҘеҸЈж–№жі•пјҡжҢүеұӮпјҲBFSпјүз»„иЈ…зӣ®еҪ•дёҺж–Үд»¶е®һдҪ“пјҢжұҮжҖ»еҫ…е…Ҙеә“ж•°жҚ®
  • жү№йҮҸ ID йў„еҲҶй…ҚпјҡдёҖж¬ЎжҖ§з”іиҜ· ID е№¶дҝ®жӯЈзҲ¶еӯҗеј•з”Ё
  • ејӮжӯҘж¶Ҳиҙ№иҖ…пјҡдёӢиҪҪгҖҒдёҠдј дёҺиҝӣеәҰи·ҹиёӘ
  • е·Ҙе…·ж–№жі•пјҡдёӢиҪҪе№¶дёҠдј пјҲеёҰиҝӣеәҰеӣһи°ғпјүгҖҒиҝӣеәҰиҫ“е…ҘжөҒе°ҒиЈ…

е…ій”®зӮ№и§ЈиҜ»пјҲBFS + жү№йҮҸе…Ҙеә“пјү

  • йҳҹеҲ—еҲҶеұӮйҒҚеҺҶпјҡDeque<FolderLevel> дҝқиҜҒжҢүеұӮеӨ„зҗҶпјҢfoldersByLevel д»Ҙж·ұеәҰдёә key иҒҡеҗҲеҗҢеұӮ FolderEntityпјҢдҫҝдәҺеҗҺз»ӯеҲҶеұӮе…Ҙеә“гҖӮ
  • ж №еұӮе…је®№еӨҡзұ»еһӢпјҡж—ўж”ҜжҢҒзӣҙжҺҘж–Үд»¶пјҲз«ӢеҚіжһ„е»ә DatasetEntityпјүпјҢд№ҹж”ҜжҢҒзҹҘиҜҶеә“/ж–Үд»¶еӨ№пјҲжһ„е»ә FolderEntity е…ҘйҳҹпјүгҖӮ
  • еӯҗиҠӮзӮ№иЈ…й…Қпјҡж–Үд»¶вҶ’DatasetEntityпјӣж–Үд»¶еӨ№вҶ’FolderEntity е№¶е…ҘйҳҹпјҢжҢҒз»ӯ BFSгҖӮ
  • жү№йҮҸйў„еҲҶй…Қ IDпјҡйҖҡиҝҮ sequenceMapper.next*Batch(n) иҺ·еҸ–иҝһз»ӯеҢәй—ҙпјҢйҒҝе…Қе№¶еҸ‘з«һдәүпјӣз”Ё tmpIdMap дҝ®жӯЈ parentId дёҺ folderId зҡ„дёҙж—¶еҚ дҪҚеј•з”ЁгҖӮ
  • еҲҶеұӮе…Ҙеә“пјҡsaveToDatabaseByLevel еҸҜд»ҘзЎ®дҝқзҲ¶еұӮе…Ҳе…Ҙеә“пјҲж»Ўи¶іеӨ–й”®/еұӮзә§зәҰжқҹпјүпјҢеҗҢж—¶жү№йҮҸеҶҷжҸҗеҚҮеҗһеҗҗгҖӮ

ејӮжӯҘж¶Ҳиҙ№иҖ…пјҡдёӢиҪҪ + дёҠдј  + иҝӣеәҰ

иҜҙжҳҺпјҡ

  • е№ӮзӯүдёҺзҠ¶жҖҒжңәпјҡupdateStatusIfMatch(READY вҶ’ DOWNLOADING) дҪңдёәвҖңд№җи§Ӯй”ҒвҖқй—Ёй—©пјҢиҝҮж»ӨйҮҚеӨҚжҠ•йҖ’гҖӮ
  • иҝӣеәҰи·ҹиёӘпјҡд»»еҠЎејҖе§ӢеңЁ Redis еҶҷе…Ҙ percent=0пјҢдёӢиҪҪ/дёҠдј иҝҮзЁӢдёӯжҢҒз»ӯиҰҶзӣ–жӣҙж–°пјӣе®ҢжҲҗжҲ–еӨұиҙҘеҗҺжё…зҗҶ KeyгҖӮ
  • дҝЎжҒҜиЎҘе…ЁпјҡиӢҘж¶ҲжҒҜдёӯж—  fileUrl/fileSizeпјҢе…ҲйҖҡиҝҮиҝңз«ҜжҺҘеҸЈиЎҘе…ЁпјҢдҝқиҜҒеҗҺз»ӯжөҒзЁӢеҒҘеЈ®гҖӮ
  • зЎ®и®Өзӯ–з•ҘпјҡжҲҗеҠҹ basicAckгҖҒеӨұиҙҘ basicNack(requeue=false)пјҢйҒҝе…Қжӯ»еҫӘзҺҜпјӣдёҡеҠЎй”ҷиҜҜеҗҢж—¶жӣҙж–°ж•°жҚ®йӣҶзҠ¶жҖҒдёә ERRORгҖӮ

дёӢиҪҪе№¶дёҠдј е®һзҺ°пјҲеёҰиҝӣеәҰеӣһи°ғпјү

иҜҙжҳҺпјҡ

  • зӣҙиҝһдёӢиҪҪпјҡжӢјжҺҘеҸ—жҺ§дёӢиҪҪ URLпјҢжҗәеёҰйүҙжқғеӨҙпјӣз»ҹдёҖ 30s иҝһжҺҘ/йҰ–еҢ…и¶…ж—¶гҖӮ
  • иҝӣеәҰеҢ…иЈ…пјҡиӢҘе·ІзҹҘж–Үд»¶еӨ§е°ҸпјҢеҲҷз”Ё ProgressInputStream и®Ўз®—зҷҫеҲҶжҜ”е№¶еӣһи°ғпјӣжңӘзҹҘеӨ§е°ҸеҲҷзӣҙжҺҘйҖҸдј гҖӮ
  • дёҠдј иҗҪең°пјҡйҖҡиҝҮ minioTemplate.putObject еҶҷе…Ҙзӣ®ж ҮжЎ¶пјӣз»“жқҹеҗҺе…ңеә•еӣһи°ғ 100%гҖӮ

иҝӣеәҰжөҒе°ҒиЈ…пјҡProgressInputStream

иҜҙжҳҺпјҡ

  • иҪ»йҮҸе°ҒиЈ…пјҡиҰҶзӣ– read/read(byte[],...)пјҢзҙҜи®Ўеӯ—иҠӮж•°гҖӮ
  • иҠӮжөҒеӣһи°ғпјҡжҜҸ вүҘ200ms жҲ–иҜ»ж»Ўж—¶и§ҰеҸ‘дёҖж¬ЎпјҢйҒҝе…Қйў‘з№Ғи°ғз”ЁйҖ жҲҗжҠ–еҠЁдёҺиҙҹиҪҪгҖӮ
  • зІҫеәҰи¶іеӨҹпјҡд»Ҙ totalBytes и®Ўз®—зҷҫеҲҶжҜ”пјҢж»Ўи¶іеүҚз«ҜиҝӣеәҰжқЎеұ•зӨәйңҖжұӮгҖӮ

и®ҫи®ЎжҖқжғідёҺеҸ–иҲҚ

  • и§ЈиҖҰдёҺеј№жҖ§пјҡеҜје…Ҙе»әжЁЎдёҺеӨ§ж–Үд»¶дј иҫ“еҪ»еә•и§ЈиҖҰпјҢеҶҷеә“еҗҺйҖҡиҝҮ MQ ејӮжӯҘжҺЁиҝӣпјҢжҸҗеҚҮеҸҜз”ЁжҖ§дёҺеҸҜи§ӮжөӢжҖ§гҖӮ
  • е№ӮзӯүдёҺдёҖиҮҙжҖ§пјҡзҠ¶жҖҒи·іеҸҳйҮҮз”Ё CAS иҜӯд№үпјҲREADYвҶ’DOWNLOADINGвҶ’DONE/ERRORпјүпјҢйҳІжӯўйҮҚеӨҚж¶Ҳиҙ№дёҺд№ұеәҸжӣҙж–°гҖӮ
  • жү№йҮҸеҢ–пјҡID жү№йҮҸеҲҶй…ҚдёҺеҲҶеұӮжү№йҮҸе…Ҙеә“еҮҸе°‘еҫҖиҝ”пјҢжҸҗеҚҮеҗһеҗҗе№¶ејұеҢ–зғӯзӮ№з«һдәүгҖӮ
  • иө„жәҗжҺ§еҲ¶пјҡдёӢиҪҪжөҒејҸиҪ¬еҸ‘иҮіеҜ№иұЎеӯҳеӮЁпјҢйҒҝе…Қжң¬ең°иҗҪзӣҳдёҺиҝҮеәҰеҶ…еӯҳеҚ з”ЁгҖӮ
  • жңүжҚҹйҮҚиҜ•зҡ„жқғиЎЎпјҡж¶Ҳиҙ№еӨұиҙҘ nack(requeue=false) дәӨз”ұжӯ»дҝЎ/жҠҘиӯҰеӨ„зҗҶпјҢйҒҝе…ҚжҜ’ж¶ҲжҒҜж— йҷҗйҮҚиҜ•гҖӮ

йЈҺйҷ©дёҺж”№иҝӣе»әи®®

  • д»»еҠЎеҸҜжҒўеӨҚжҖ§пјҡеј•е…Ҙд»»еҠЎиЎЁи®°еҪ•зҠ¶жҖҒдёҺеӨұиҙҘеҺҹеӣ пјҢй…ҚеҗҲиЎҘеҒҝд»»еҠЎпјҲйҮҚиҜ•/з»ӯдј пјүгҖӮ
  • еӨ§ж–Үд»¶ж–ӯзӮ№з»ӯдј пјҡеҜ№жҺҘеҜ№иұЎеӯҳеӮЁзҡ„еҲҶзүҮ/ж–ӯзӮ№иғҪеҠӣпјҢеӨұиҙҘеҸҜд»ҺжңҖиҝ‘еҲҶзүҮз»§з»ӯгҖӮ
  • зӣ®еҪ•ж·ұеәҰдёҺи§„жЁЎпјҡйҷҗеҲ¶жңҖеӨ§ж·ұеәҰдёҺиҠӮзӮ№жҖ»йҮҸпјҢйҳІжӯўжҒ¶ж„Ҹ/ејӮеёёж•°жҚ®еҜјиҮҙй•ҝж—¶й—ҙеӨ„зҗҶгҖӮ
  • йҖҹзҺҮдёҺе№¶еҸ‘пјҡдёӢиҪҪ/дёҠдј е№¶еҸ‘ж•°й…ҚзҪ®еҢ–пјӣиҝӣеәҰеӣһи°ғиҠӮжөҒзӘ—еҸЈеҸҜи°ғгҖӮ
  • е®үе…ЁжҖ§пјҡдёӢиҪҪ URL йҮҮз”Ёдёҙж—¶зӯҫеҗҚпјӣжңҖе°Ҹжқғйҷҗзҡ„еҜ№иұЎеӯҳеӮЁеҮӯиҜҒпјӣж•Ҹж„ҹж—Ҙеҝ—и„ұж•ҸгҖӮ
  • еҸҜи§ӮжөӢпјҡеўһеҠ  Prometheus жҢҮж ҮпјҲйҳҹеҲ—ж»һз•ҷгҖҒжҲҗеҠҹзҺҮгҖҒP95/P99 ж—¶е»¶гҖҒеёҰе®ҪеҲ©з”ЁпјүгҖӮ

жҖ»з»“

  • дҪҝз”Ё BFS дҝқйҡңвҖңжҢүеұӮжһ„е»әгҖҒжҢүеұӮе…Ҙеә“вҖқпјҢз»“еҗҲжү№йҮҸ ID йў„еҲҶй…ҚпјҢе®һзҺ°й«ҳж•Ҳдё”з»“жһ„зЁіе®ҡзҡ„еҜје…ҘгҖӮ
  • йҖҡиҝҮ MQ и§ЈиҖҰй•ҝиҖ—ж—¶зҡ„дёӢиҪҪ/дёҠдј пјҢе№¶д»Ҙ Redis е®һж—¶и®°еҪ•иҝӣеәҰпјҢз”ЁжҲ·еҸҜи§ҒжҖ§дёҺзі»з»ҹйІҒжЈ’жҖ§е…јйЎҫгҖӮ
  • и®ҫи®ЎеңЁе№ӮзӯүгҖҒжү№йҮҸеҢ–гҖҒиө„жәҗиҠӮеҲ¶дёҠеҒҡеҮәжқғиЎЎпјҢеҗҺз»ӯеҸҜеӣҙз»•еҸҜжҒўеӨҚжҖ§дёҺеҸҜи§ӮжөӢжҖ§жҢҒз»ӯдјҳеҢ–гҖӮ
